# Jungle Studio

Plateforme interne dédié à réunir les LLM

## Objectif
Centraliser les outils, agents IA et workflows liés aux projets.

## Stack technique
- Frontend : React + Vite
- Backend : Supabase
- Auth : Supabase Auth
- Database : PostgreSQL (Supabase)
- Storage : Supabase Storage
